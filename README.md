# SpectralSort

Спектральная сортировка массива чисел на основе вектора Фидлера графа Лапласа.

## Алгоритм

1. Строится матрица весов `W[i][j] = exp(-|a[i] - a[j]|)` для всех пар элементов.
2. По матрице весов формируется лапласиан графа.
3. Методом степенных итераций вычисляется вектор Фидлера — собственный вектор, соответствующий второму наименьшему собственному значению лапласиана.
4. Элементы массива сортируются по координатам вектора Фидлера.

## Пример

```
Введите количество элементов: 5
Введите элементы массива через пробел: 3.1 1.2 4.5 2.0 0.9
SpectralSort: 0.9 1.2 2.0 3.1 4.5
```
